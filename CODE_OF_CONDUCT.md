# Code of Conduct

## Our Standards

We aim for a respectful, inclusive, and technical collaboration culture.

Expected behavior:
- Be respectful and constructive.
- Focus on facts and reproducible evidence.
- Assume good intent.
- Accept and provide feedback professionally.

Unacceptable behavior:
- Harassment, discrimination, or personal attacks.
- Public disclosure of unpatched vulnerabilities.
- Deliberate disruption of project collaboration.

## Enforcement

Project maintainers may remove comments, close discussions, or block participation for violations.

## Scope

This policy applies to:
- issues
- pull requests
- discussions
- other official project communication channels
